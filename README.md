# Documentación técnica (PROYECTO EN DESARROLLO)

## STACK

Para el backend: node.js, express.js, mongoDB y socket.io

- `npm init -y`
- `npm i express mongoose dotenv jsonwebtoken bcryptjs cookie-parser cloudinary socket.io`
- `npm i nodemon -D`

En `package.json` añadir script `"dev": nodemon index.js` y `"type": "module"`

Para el frontend: react.js, tailwindCSS, zustand y daisy UI

- `npm create vite@latest .`
- `npm install`
